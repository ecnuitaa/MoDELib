Outputs DislocationNetwork data to the following files (x is the runID): ./E/E_x.txt (DislocationSegment(s) are always outputted) ./V/V_x.txt (DislocationNode(s) are always outputted) ./C/C_x.txt (DislocationCell(s) only if outputSpatialCells==true) ./G/G_x.txt (GlidePlane(s) only if outputGlidePlanes==true) ./P/P_x.txt (PK forces only if outputPKforce==true) ./W/W_x.txt (elastic energy only if outputElasticEnergy==true) ./D/D_x.txt (mesh displacement only if outputMeshDisplacement==true)

3- Outputs the nearest neighbor Cell structures to file C_*.txt where * is the current simulation step

4- Outputs the glide planes
