template<typename ParticleType, short unsigned int dim>
struct model::SpatialCell< ParticleType, dim >

A dim-dimensional cell occupying the spatial region cellID <= x/cellSize < (cellID+1). SpatialCell is aware off all ParticleType objects present inside it.
