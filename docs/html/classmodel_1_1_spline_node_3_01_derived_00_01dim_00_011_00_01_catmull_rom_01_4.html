Definition at line 126 of file SplineNode_CatmullRom.h.

template<typename Derived , short unsigned int dim>
void model::SplineNode< Derived, dim, 1, CatmullRom >::set ( const VectorDim &  P_in)
inline

Because of the CatmullRom rule for parametric tangents, changing the position of this CatmullRom node must also change its parametric tangent and the parametric tangents of its first-neighboring nodes. This in turn affects the shape of all SplineSegments attached to the first-neighboring nodes. The update strategy is as follows:

1- Sets P=P_in

2- Transmits 'make_T' on the neighbor nodes of level (corder+1=2), that is this node and its first-neighbors
