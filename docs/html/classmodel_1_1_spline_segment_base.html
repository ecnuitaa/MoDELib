Detailed Description

template<typename Derived, short unsigned int dim, short unsigned int corder>
class model::SplineSegmentBase< Derived, dim, corder >

A SplineSegmentBase is a special kind of parametric curve where the position vector can be written as: 

r(u)=N(u)*q

where N(u) is a (dim x NdofXsegment) matrix of shape functions and q is a (NdofXsegment x 1) column vector of degrees of freedom, where NdofXsegment= dim x n and n=porder+1 is the number of shape functions: 

| N1  0  0              Nn  0  0 |
N= |  0 N1  0   ...  0 Nn  0 |
|  0  0 N1               0  0 Nn |

Each shape function is a porder: 

[N1 ... Nn]= [1 u ... n^porder] * SFC

where SFC is a n x n matrix of shape function coefficients that in general dependes on the knot vector. It's posible to take advantage of this particular form of shape functions to calculate ruu and ru therefore these functions are redefined here (virtual in ParametricCurve) and the flow chart of their implementation is shown below:

^                                 ^                             ^
|                                 |                             | 
make_SFuu()       make_SFu()    make_SF()
^                                 ^                             ^
|                                 |                             |
make_Nuu()                make_Nu()             make_N()
^                                 ^                             ^
|                                 |                             |
make_ruu()                make_ru()             make_r()        (this level is virtual in ParametricCurve)
