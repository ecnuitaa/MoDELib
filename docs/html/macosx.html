Updating gcc

At the time of writing this guide (May 2013), the version of gcc that ships with Xcode is 4.2. You need version 4.7 or higher to compile the c++11 features used in MODEL. Therefore you need to install a more recent version of gcc in your Mac OsX.

You can install the most recent gcc using MacPorts.

After you have successfully installed the current version of gcc, make sure to select that version as the default gcc compiler. You can select a particular version of gcc by first installing gcc_select, again via MacPorts: 

sudo port install gcc_select

Then, list the available gcc versions in your system: 

port select --list gcc

And finally select the most current version, say mp-gcc47: 

sudo port select --set gcc mp-gcc47

Make sure to complete these steps before installing open-MPI and METIS.

Porting other Unix/Linux packages

MacPorts can also be used to port the following packages to Mac OsX:

  • ImageMagick
  • FFmpeg
