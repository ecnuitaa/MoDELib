\item Use Intel 13.\+0 compiler

Add the line to the bottom of $\sim$/.bashrc\+:

download \href{http://www.open-mpi.org/software/ompi/v1.6/downloads/openmpi-1.6.3.tar.gz}{\tt http\+://www.\+open-\/mpi.\+org/software/ompi/v1.\+6/downloads/openmpi-\/1.\+6.\+3.\+tar.\+gz} (or whatever the latest stable release).

\item Metis
